Inherits from NSObject
Conforms to MOCASerializable
Declared in MOCAPropertyContainer.h

Overview

Property container.

Properties are identified by a string name. Values are restricted to:

  • NSNull
  • NSString
  • NSNumber (bool, int, long long, double and float)
  • NSDate
